Política de Privacidade

Última atualização: 19/04/2026

Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações ao utilizar nosso bot e serviços associados.

1. Informações Coletadas

  • ID de usuário do Discord
  • ID de servidor (guild)
  • ID de canais
  • Dados de uso de comandos

2. Uso das Informações

  • Garantir o funcionamento do bot
  • Melhorar funcionalidades
  • Prevenir abusos e fraudes

3. Compartilhamento de Dados

  • Não vendemos ou compartilhamos dados com terceiros
  • Dados podem ser divulgados apenas se exigido por lei

4. Armazenamento de Dados

  • Os dados são armazenados de forma segura
  • Retidos apenas pelo tempo necessário

5. Segurança

Adotamos medidas razoáveis para proteger as informações contra acesso não autorizado, alteração ou destruição.

6. Direitos do Usuário

  • Solicitar exclusão de dados
  • Solicitar informações armazenadas

7. Alterações

Esta política pode ser atualizada a qualquer momento. O uso contínuo do serviço implica aceitação das mudanças.
